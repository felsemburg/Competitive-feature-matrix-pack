# GEMINI.md

> Author: Rafael Felsemburg · Skill: Competitive Feature Matrix v1.0

This project includes the Competitive Feature Matrix skill.

When the user asks to compare competitor features, run a feature gap analysis,
decide what to build next, or rank features across competing products into a
decision matrix, read and follow:

`skill/competitive-feature-matrix/SKILL.md` (plus its `references/` files).

Mandatory: ask the scoping questions first; present the discovered competitor
list and WAIT for the user to approve or edit it before analyzing features;
output the full marked matrix with a plain-language "how to read this" block and
the decision view; and force any user-named feature into the matrix as its own
row even at 0% adoption. Read the file rather than working from memory — the
scoring rubric and format are precise.
